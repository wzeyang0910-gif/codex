# 原研获客平台局域网使用说明

## 启动与停止

1. 在主机桌面双击“启动原研获客网站”。
2. 保持启动窗口开启。
3. 需要停止网站时，在启动窗口按 `Ctrl+C`，或直接关闭该窗口。

网站不会随 Windows 自动启动。

## 访问地址

公司局域网内优先使用：

```text
http://LAPTOP-AN04BPA1:3000
```

如果电脑名称无法解析，可使用当前局域网地址：

```text
http://192.168.69.47:3000
```

局域网 IP 可能由路由器重新分配，电脑名称地址通常更稳定。网站只允许当前私有局域网访问 TCP 3000 端口，不应把该端口映射到互联网。

## 登录账号

系统内置管理员和业务员角色。账号及密码由内部管理员分发，不要把密码写入群聊、公开文档或邮件。

## 自动备份

- Windows 每天凌晨 `02:00` 自动备份 PostgreSQL 数据库。
- 本机备份目录为 `D:\Codex\原研\web\backups`。
- 本机备份不能防止整机或硬盘损坏，应定期复制到另一块磁盘或受控的公司存储。

## 使用条件

- 主机必须开机并连接公司局域网。
- 网站启动窗口必须保持开启。
- 启动脚本会在网站运行期间阻止因空闲而自动睡眠，停止网站后恢复原来的电源行为。
- 合上笔记本电脑屏幕或手动选择睡眠仍可能中断网站，使用期间应保持主机开启。
- PostgreSQL 服务已设置为随 Windows 自动运行。

## 快速检查

主机可访问以下地址确认服务状态：

```text
http://127.0.0.1:3000/api/health
```

正常结果应显示 `status: ok` 和 `database: ok`。
